                            Emphasized text (em)
                Menunjukan bagian kata yang perlu di tekankan

Contoh

Oding adalah seorang pelajar

Dia adalah seorang pelajar

                    

                        Important text (strong)

Contoh

Kesehatan merupakan hal yang penting, jaga pola makan yang teratur dan jangan sampai makan tengah malam!

                    

                        Short quotation/kutipan pendek (q)

Contoh

Jika kita salah maka sebaiknya kita Meminta Maaf

                    

                        Citatiom / Rujukan pada sebuah dokumen (cite)

Contoh

Informasi selengkapnya kontak saya di Likendin Linkedin.com

                    

                        Defining terms (term)
                        Mendefinisikan sebuah istilah

Contoh :

Belajar Web adalah pembelajaran untuk menjkadi seorang web programin. Belajar Web saya lakukan untuk bekal KP

                    

                        Subscript dan super script
                        (sub di bwah text) (sup di atas text)

Contoh

Sub

Rumus molekul C22 O11

Sup

Bilangan pangkat 112 126 A23

                    

                        High light (mark)

Contoh

Lihat ini Ini adalah mark text 

                    

                        Line break (br)

Contoh

Menggunakan
Baris 1
Baris 2
Baris 3
Baris 4
